## 🚀 Comienza en 3 Simples Pasos

```bash
# 1. Instala las dependencias
npm install

# 2. Configura la base de datos (crea la BD y carga datos de ejemplo)
npm run setup

# 3. Inicia la aplicación
npm run dev
```

Luego abre http://localhost:3000 en tu navegador.

**Eso es todo.** 3 comandos y estás corriendo.

## ❓ Preguntas Frecuentes

**P: ¿Necesito tener una cuenta online para usar esto?**
A: No. Funciona 100% local en tu taller.

**P: ¿Se perderán mis datos si apago la computadora?**
A: No. Los datos se guardan en el archivo `dev.db` (SQLite).

**P: ¿Puedo llevar a la nube después?**
A: Sí. Es fácil migrar de SQLite a PostgreSQL en AWS, Vercel, etc.

**P: ¿Cuánto tiempo tarda el setup?**
A: 10 segundos con `npm run setup`.

**P: ¿Puedo personalizar el diseño?**
A: Claro. Está hecho con Tailwind CSS y shadcn/ui, muy fácil de modificar.
